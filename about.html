Github Treemaps

This is an experiment to visualize Github repositories as treemaps.

How It Works

There are two distinct components involved in creating these treemaps. The user interface is written in JavaScript using AngularJS, and the treemaps are rendered with D3.js. The front end is cabable of rendering specially formatted JSON files that are generated from trees and commits of Github repositories. Currently commits, repository tree, and programming languages of a repository are visualized as treemaps. Some of these views are zoomable.

A backend service is in charge of pulling Github repositories and generating the statistics. It is written in Python and is using Celery, pygit2, Flask, and SQLAlchemy.

Pulling and exploring Github repositories is not a light task. The backend is exploiting Amazon EC2 Spot Instances to process repositories. Still it is not realistic to summarize all repositories. Currently the most starred and forked repositories are automatically kept updated. In addition, there is a ability to queue a repository for indexing if not already indexed.

All the source code is available at Github. If you run into problems, please leave a bug report on Github issues.
